Umphrey's McGee Guitar Tablature

Welcome to a repository of guitar transcriptions to many of the songs played by Umphrey's McGee.

Official transcriptions can be purchased from Umphrey's McGee Merch.

If you have a correction or new submission, please add or edit the file directly in Github or open an issue.

How to contribute

To contribute, you'll need to create a Github account, which is free.

Edit a transcription

  1. Browse to the file you want to edit (or find it using the search box at the top) and click the "Edit" button near the top of the file.
  2. Make your changes, then click "Propose File Change."
  3. On the next page, click "Send pull request" to send a request to have your changes merged back into the main project. If you've changed your mind and want to delete your changes, click on the "Branches" tab and delete the branch.
  4. Once your request has been accepted, you can delete the old branch (optional).

Create a new transcription

  1. For the sake of simplicity, use the links below:
    • New transcription
  2. Paste or type in the transcription, then click "Propose new file"
  3. On the next page, click "Send pull request" to send a request to have your transcription file merged into the main project.
